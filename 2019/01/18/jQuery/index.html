#jQuery

jQuery是js的一个库，封装了我们开发过程中常用的一些功能
代替原生js

###使用jQuery

###jQuery 和原生 Dom 之间转换

  • Dom转换jQuery –> $(Dom元素)
  • jQuery转换Dom –> $(btnObj).get[0]

###1.5.1 jQuery选择器

  • ####基本选择器

    • id选择器 $(‘#box’)
    • 标签选择器 $(‘div’)
    • 类选择器 $(‘.cls’)

  • ####层级选择器

    • 后代选择器 $(‘#box li’)
    • 子代选择器 $(‘#box>li’)

  • ####索引选择器

    • :eq(index)
      $(“li:eq(2)”) // 选择li元素中索引号为2的一个元素
    • :odd
      $(“li:odd”) // 选择li元素中索引号为奇数的所有元素
    • :even
      $(“li:odd”) // 选择li元素中索引号为偶数
    • :gt(index)
      $(“li:eq(2)”) // 选择li元素中索引号大于2的一个元素
    • :lt(index)
      $(“li:eq(2)”) // 选择li元素中索引号小于2的一个元素

  • ####选择元素

    • find()
      $(“#box”).find(“li”) // 查找指定元素的所有后代元素（子子孙孙）
    • children()
      $(“#box”).children(“ul”) 查找指定元素的直接子元素（亲儿子元素）
    • siblings()
      $(“#box”).siblings() // 查找所有兄弟元素（不包括自己）
    • index()
      var index = $(this).index() // 获取指定元素的索引号（从0开始的）

###属性

  • text()

    • $(“#box”).text() // 获取文本
    • $(“#box”).text(“文本”) // 设置文本

  • html()

    • $(“#box”).html() // 获取文本
    • $(“#box”).html(“文本”) // 设置文本

  • val()

    • $(“#box”).val() // 获取元素内容
    • $(“#box”).val(“文本”) // 设置元素内容

  • css()

    • 获取样式值
      $(#box).css(“样式名”)
    • 设置一个样式
      $(#box).css(“样式名”，”样式值”)
    • 设置多个样式
      $(#box).css({ “样式名1”：”样式值1”，”样式名2”，”样式名2”})

  • attr()

    设置或返回被选元素的属性和值
    attr() 值只可以为 string

    1. 返回属性的值
      $(selector).attr(属性)
    2. 设置属性和值
      $(selector).attr(属性,值)
    3. 使用函数设置属性和值
      $(selector).attr(属性,function(index,值))

  • prop()

    设置或返回被选元素的属性和值
    prop() 值可以是 string 也可以是 arr

    1. 返回属性的值
      $(selector).prop(属性)
    2. 设置属性和值
      $(selector).prop(属性,值)
    3. 使用函数设置属性和值
      $(selector).prop(属性,function(index,值))

  • addClass()

    给元素添加类名,以前的class依然存在

$(selector).addClass(类名)

  • removeClass()

    删除元素指定类名

$(selector).removeClass(类名)

  • toggleClass

    如果元素有指定类则会删除；如果没有指定类则会添加

$(selector).toggleClass(类名)

###文档处理

  • append() appendTo()
    被选中 元素内部 后面/前面 添加指定内容
    $(父元素).append(需要插入内容) $(需要插入内容).appendTo(父元素)

  • prepend() prependTo()
    被选中 元素内部 后面/前面 添加指定内容
    $(父元素).prepend(需要插入内容)

  • after() before()
    被选中元素 结尾/前面 添加指定内容
    $(元素).after(需要插入内容) $(元素).before(需要插入内容)

    • insertAfter() insertBefore()
      把所有匹配的元素插入到另一个、指定的元素元素集合的后面/前面
      $("p").insertAfter("#foo") $("p").insertBefore("#foo")

    • replaceWith()
      将所有匹配的元素替换成指定的HTML或DOM元素。
      $("p").replaceWith("<b>Paragraph. </b>");

    • replaceAll()
      用匹配的元素替换掉所有 selector匹配到的元素。
      $("<b>Paragraph. </b>").replaceAll("p");
