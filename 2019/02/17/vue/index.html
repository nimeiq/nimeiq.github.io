Vue

不再操作DOM，Vue.js是一套构建用户界面的框架

MVC(Model View Controller)

M:数据层
V:视图层
C:控制层(业务逻辑)

MVVM

  • 前端视图层的分层开发思想，把每个页面分开;
  • 提供数据的双向绑定
  • VM是核心部分，处理数据双向绑定

M: 保存每个页面单独的数据 –> VM将数据存储在M中
VM: 调度者，分割M和V –> VM从M中获取数据；发送给V
V: 每个页面中的HTML结构 –> 将数据存储在VM中

###Vue的使用

在全局下载 npm i Vue Vue-cli -g

用脚手架创建项目

  • vue create “项目名称”
  • cd 到项目根目录下
  • vue run server 运行

使用

生命周期函数

创建

beforeCreate(){}
data和methods未被初始化
8个周期函数创建完成后执行

created(){}
data和methods都已经初始化
如果有页面加载完成就要执行的函数就在此执行

beforeMount(){}
表示模板在内存中编译成功，但未渲染到页面上

mounted(){}
表示内存中的模板已经挂载到页面中，可以操作DOM

运行

当data数据改变时才会执行，最少执行0次

beforeUpdate(){}
数据改变，页面没有改变

updated(){}
页面数据和data数据保持一致，页面上是最新的数据

销毁

beforeDestroy(){}
当执行此函数，Vue实例就已经从运行到销毁阶段
data、methods、过滤器、指令 处于可用状态，还没有真正被销毁

destroyed(){}
组件已经完全被销毁
data、methods、过滤器、指令 不可用
